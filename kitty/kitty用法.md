# 常用命令

1. 对比两个文件：`kitty +kitten diff file1.txt file2.txt`
2. 对比两个目录（递归）：`kitty +kitten diff dir1/ dir2/`

# 快捷键

## 1. 标签页管理 (Tab Management)

| **动作 (Action)**                      | **默认快捷键 (Linux)**         |
| -------------------------------------- | ------------------------------ |
| **新建标签页** (New tab)               | `Ctrl` + `Shift` + `t`         |
| **关闭标签页** (Close tab)             | `Ctrl` + `Shift` + `q`         |
| **下一个标签页** (Next tab)            | `Ctrl` + `Shift` + `Right`     |
| **上一个标签页** (Previous tab)        | `Ctrl` + `Shift` + `Left`      |
| **切换布局** (Next layout)             | `Ctrl` + `Shift` + `l`         |
| **向后移动标签页** (Move tab forward)  | `Ctrl` + `Shift` + `.`         |
| **向前移动标签页** (Move tab backward) | `Ctrl` + `Shift` + `,`         |
| **设置标签页标题** (Set tab title)     | `Ctrl` + `Shift` + `Alt` + `t` |

## 2. 剪贴板(Clipboard)

| **动作 (Action)**                       | **默认快捷键 (Linux)**         |
| --------------------------------------- | ------------------------------ |
| **复制到剪贴板** (Copy to clipboard)    | `Ctrl` + `Shift` + `c`         |
| **从剪贴板粘贴** (Paste from clipboard) | `Ctrl` + `Shift` + `v`         |
| **从选中区粘贴** (Paste from selection) | `Ctrl` + `Shift` + `s`         |
| **增大字号** (Increase font size)       | `Ctrl` + `Shift` + `=`         |
| **减小字号** (Decrease font size)       | `Ctrl` + `Shift` + `-`         |
| **重置字号** (Restore font size)        | `Ctrl` + `Shift` + `Backspace` |


## 3. 窗口管理 (Window Management)

| **动作 (Action)**                        | **默认快捷键 (Linux)**         |
| ---------------------------------------- | ------------------------------ |
| **新建窗口** (New window)                | `Ctrl` + `Shift` + `Enter`     |
| **新建 OS 窗口** (New OS window)         | `Ctrl` + `Shift` + `n`         |
| **关闭窗口** (Close window)              | `Ctrl` + `Shift` + `w`         |
| **下一个窗口** (Next window)             | `Ctrl` + `Shift` + `]`         |
| **上一个窗口** (Previous window)         | `Ctrl` + `Shift` + `[`         |
| **向前移动窗口** (Move window forward)  | `Ctrl` + `Shift` + `f`         |
| **向后移动窗口** (Move window backward) | `Ctrl` + `Shift` + `b`         |
| **置顶窗口** (Move window to top)        | ctrl+shift+\` |
|   **视觉聚焦窗口** (Visually focus window)	|`Ctrl + Shift + F7`|
| **视觉交换窗口** (Visually swap window)  | `Ctrl` + `Shift` + `F8`        |
| **切换全屏** (Toggle fullscreen)        | `Ctrl` + `Shift` + `F11`       |
| **切换最大化** (Toggle maximized)       | `Ctrl` + `Shift` + `F10`       |
| **聚焦特定窗口** (Focus specific window)	|`Ctrl + Shift + 1 ... 0`|
| **调整窗口大小**	|`Ctrl + Shift + r`|
